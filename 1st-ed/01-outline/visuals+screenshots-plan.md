# 🎯 Implementation Notes

- Screenshots should be **clean, uncluttered, with sensitive info blurred**.
- Where tool UIs change frequently (e.g., ChatGPT, Canva), use **conceptual mock-ups** instead of literal screenshots for longevity.
- Infographics can be built in **Canva or Figma**, ensuring consistent style (same font/colors across book).
- Aim for **1–2 visuals per chapter page spread** → keeps the book practical but not overloaded.
