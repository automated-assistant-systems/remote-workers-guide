# **Quick Win:** How to Use This Library

1. Copy a prompt, then **add context** (paste text, describe your goal).
2. Always **review and edit outputs** — AI is a draft partner, not a final authority.
3. Build your own “personal library” of modified prompts as you discover what works best for you.
