# Visuals Plan
The Remote Worker’s Survival Guide in the AI Age  
Practical Tools, Workflows, and Strategies for Thriving Anywhere

This document defines the visual strategy for the manuscript.

Its purpose is to ensure that visuals support the book’s architecture, reinforce key concepts, and remain consistent across PDF, EPUB, and web formats.

Visuals should clarify ideas, not decorate pages.

---

# Visual Strategy Principles

## 1. Prefer Conceptual Diagrams Over Screenshots

Use visuals that explain systems, workflows, and relationships.

Preferred visual types:

- workflow diagrams
- process maps
- decision filters
- comparison frameworks
- progression models

Avoid interface screenshots unless they are essential.

---

## 2. Each Visual Must Have a Distinct Job

Do not include multiple visuals that communicate the same idea in slightly different forms.

Each visual should answer a specific reader question, such as:

- What changed?
- Where does the problem come from?
- How does the workflow operate?
- How do I choose between options?
- What does implementation look like?

---

## 3. Visuals Must Be Introduced in the Text

Every visual should be introduced before it appears.

Recommended pattern:

- introduce the concept
- present the visual
- briefly interpret what the reader should notice

---

## 4. Visuals Should Be Durable Across Formats

Visuals should remain understandable in:

- PDF
- EPUB
- web publishing

This means:

- minimal dependence on tiny text
- no interface-specific references unless necessary
- clear labels
- simple shapes and hierarchy

# Production Notes

When preparing visuals for production:

- use consistent naming conventions
- keep typography readable at ebook scale
- avoid dense text inside visuals
- prefer labels and simple directional flow
- ensure each visual has a caption and short interpretation

Suggested file naming pattern:

visual-01-remote-work-before-vs-after-ai
visual-02-remote-work-evolution-timeline
visual-03-ai-productivity-stack

Continue sequentially through the manuscript.
