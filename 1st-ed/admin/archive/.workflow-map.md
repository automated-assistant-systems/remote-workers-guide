# .workflow-map.md

AI Editorial Workflow

Project: **The Remote Worker’s Survival Guide in the AI Age**

Purpose:
Define the **AI-assisted editing pipeline** used to produce the manuscript.

This ensures that each GPT performs a specific role and prevents overlap.

---

# Editorial Pipeline

The book is produced using the following workflow.

Book Architect GPT
↓
Restructure & Rewrite GPT
↓
Visual Layout Integrator GPT
↓
Manuscript Editor GPT
↓
Fact Checker GPT
↓
Reader Clarity Reviewer GPT

---

# 1. Book Architect GPT

Role:

Structural editor and manuscript architect.

Responsibilities:

• analyze manuscript architecture
• design part structure
• identify redundancies
• recommend chapter restructuring
• define narrative progression

Output:

Structural blueprint.

---

# 2. Restructure & Rewrite GPT

Role:

Implementation editor.

Responsibilities:

• reorganize chapters
• rewrite sections when required
• merge or split chapters
• align manuscript with architecture plan

Output:

Restructured chapters.

---

# 3. Visual Layout Integrator GPT

Role:

Visual placement editor.

Responsibilities:

• insert diagrams and screenshots
• add captions
• add interpretation text
• ensure visuals support concepts

Output:

Chapters with integrated visuals.

---

# 4. Manuscript Editor GPT

Role:

Line editor and style editor.

Responsibilities:

• improve clarity and tone
• standardize chapter structure
• remove formatting inconsistencies
• enforce Markdown rules

Output:

Clean, publishable manuscript.

---

# 5. Fact Checker GPT

Role:

Accuracy reviewer.

Responsibilities:

• verify claims
• flag unsupported statements
• recommend citations where necessary

Output:

Fact-check report and suggested corrections.

---

# 6. Reader Clarity Reviewer GPT

Role:

Reader experience editor.

Responsibilities:

• improve readability
• simplify complex explanations
• strengthen transitions
• remove ambiguity

Output:

Final clarity improvements.

---

# Build Pipeline

After editorial workflow completes, the book is built using:

scripts/build-book.sh

Outputs:

PDF

Website
